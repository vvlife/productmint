# IdeaHub 对话式交互改造计划

## 目标
将 IdeaHub 从页面式 UI 改造为类似 v0/Bolt/Lovart 的对话式交互界面，所有操作通过聊天对话框完成。

## 核心设计

### 1. 对话式主页 (`app/page.tsx`)
- 替换现有的 Timeline 列表为全屏聊天界面
- 底部固定输入框，支持文字输入
- 消息流从下往上滚动，新消息在底部
- 预设快捷操作按钮（发现需求、生成产品、社区作品）

### 2. 消息卡片类型 (`components/chat/`)

| 卡片类型 | 用途 | 交互方式 |
|---------|------|---------|
| `TextCard` | AI 回复文本 | 纯展示 |
| `IdeaCard` | 展示发现的需求 | 点击选择、查看详情 |
| `ProductCard` | 展示生成的产品方案 | 确认/调整/预览 |
| `PreviewCard` | 产品 HTML 预览 | iframe 预览、全屏 |
| `ActionCard` | 操作按钮组 | 点击执行 |
| `InputCard` | 需要用户输入 | 文本输入框 |
| `SelectCard` | 选项选择 | 点击选择 |
| `ProgressCard` | 生成进度 | 进度条展示 |
| `CopyCard` | 复制/改编产品 | 一键复制 |

### 3. 对话流程设计

```
用户: "帮我找一个AI写作相关的创业需求"
  ↓
AI: [TextCard] "正在从多个平台搜索..." [ProgressCard]
  ↓
AI: [IdeaCard x3] 展示3个相关需求，每个带"选择"按钮
  ↓
用户: 点击选择第一个
  ↓
AI: [TextCard] "好的，正在分析这个需求..." [ProgressCard]
  ↓
AI: [ProductCard] 展示AI生成的产品方案
  ↓
用户: 点击"确认生成"
  ↓
AI: [ProgressCard] "正在生成产品页面..." 
  ↓
AI: [PreviewCard] 展示产品预览 + 操作按钮
  ↓
用户: 点击"部署" / "分享" / "改编"
```

### 4. 复制/改编功能

**数据模型扩展** (`lib/types.ts`):
```typescript
interface Product {
  // ... existing fields
  clonedFrom?: string    // 来源产品ID
  clonedBy?: string[]    // 被谁复制过
}
```

**API 路由** (`app/api/clone/route.ts`):
- POST `/api/clone` - 复制产品，生成新ID，保留原HTML作为v1
- 前端显示"改编"按钮，点击后进入编辑模式

**改编流程**:
1. 用户点击"改编此产品"
2. 系统创建产品副本（clonedFrom指向原产品）
3. 用户通过对话框输入修改要求
4. AI基于原HTML和新要求生成新版本

### 5. 简易用户系统

**存储**: localStorage
```typescript
interface User {
  id: string        // user_xxx
  nickname: string  // 昵称
  avatar: string    // 头像URL或emoji
}
```

**实现**:
- 首次访问弹出昵称设置弹窗
- 昵称保存到localStorage
- 显示在对话头像中
- 用于产品归属和投票去重

### 7. 状态管理

使用 React useState + useRef 管理对话状态:
```typescript
interface ChatState {
  messages: Message[]         // 消息列表
  isTyping: boolean           // AI是否在输入
  currentFlow: string | null  // 当前对话流程
  pendingAction: any          // 待处理的操作
}

interface Message {
  id: string
  role: 'user' | 'assistant'
  content: string
  cards?: Card[]              // 附属卡片
  timestamp: string
}
```

### 8. 现有API复用

| 对话动作 | 调用API |
|---------|--------|
| 搜索需求 | `GET /api/search?q=xxx` 或 `GET /api/feed` |
| 分析需求 | `POST /api/analyze` |
| 生成产品 | `POST /api/generate` |
| 保存产品 | `POST /api/products` |
| 预览产品 | iframe 加载 `/p/{id}` |
| 部署产品 | `POST /api/deploy` |
| 复制产品 | `POST /api/clone`（新增） |
| 打榜提交 | `POST /api/submit-leaderboard` |
| 社区投票 | `POST /api/community/[id]/vote` |
